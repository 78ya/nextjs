## 项目简介

这是一个基于 **Next.js App Router** 的示例项目，集成了：

- **首页博客区块**：展示若干随机生成的博客预览卡片（仅 UI，占位数据），采用简洁线条风格的"封面图"。
- **登录页 `/login`**：使用 **Next.js Server Actions + @vercel/edge-config（Nextjs Store）** 实现的登录示例。
- **现代 UI / 暗色模式**：使用 Tailwind CSS v4 和自定义主题变量，支持深浅色切换。
- **TypeScript + ESLint**：提供基础的类型检查与代码规范。

适合作为：Next.js 16 + React 19 + Tailwind CSS 4 + Edge Config 的入门 Demo 或脚手架。
